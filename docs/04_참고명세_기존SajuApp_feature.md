# 사주 만세력 기능 명세서 (Feature Specification)

이 문서는 현재 `Saju App`에 구현된 만세력 엔진의 기능과 상세 계산 알고리즘을 기술합니다.

## 1. 사주팔자(Four Pillars) 산출

사주팔자는 생년월일시를 기준으로 연, 월, 일, 시의 4개 기둥(柱)을 세우는 것을 의미합니다.

### 1.1 연주 (Year Pillar)

* **구하는 항목**: 연간(Year Stem), 연지(Year Branch)
* **산출 방법**:
  * **기준점**: 입춘(立春, 양력 2월 4일경) 시각.
  * 입춘 이전에 태어난 경우 **전년도**의 간지를 사용합니다.
  * **계산식**:
    * 천간 Index = `(연도 - 4) % 10` (0:甲, 1:乙, ...)
    * 지지 Index = `(연도 - 4) % 12` (0:子, 1:丑, ...)

### 1.2 월주 (Month Pillar)

* **구하는 항목**: 월간(Month Stem), 월지(Month Branch)
* **산출 방법**:
  * **기준점**: 24절기(Solar Terms). (예: 2월은 입춘, 3월은 경칩 기준)
  * 해당 월의 절기 시각을 지났는지에 따라 월이 결정됩니다.
  * **월지**: 절기 기준으로 1월(寅), 2월(卯) ... 12월(丑) 매핑.
  * **월간 (월두법/오호둔갑)**: 연간에 의해 결정됩니다.
    * `월간 Index = (연간 Index * 2 + 월지 Index) % 10` (월지 Index는 寅=0 기준)
    * 예: 갑(甲)년의 인(寅)월 -> 병인(丙寅)월

### 1.3 일주 (Day Pillar)

* **구하는 항목**: 일간(Day Stem), 일지(Day Branch)
* **산출 방법**:
  * 특정 기준일(예: 1900년 1월 1일)로부터 경과한 일수를 계산하여 60갑자를 순환시킵니다.
  * `경과일수 % 60`을 통해 일주 간지를 도출합니다.
  * 야자시/조자시 구분 없이 00:00~23:59 기준으로 계산하거나, 야자시 적용 시 23:30부터 다음 날 일주를 적용하는 옵션이 있을 수 있습니다. (현재 코드는 표준시 기준)

### 1.4 시주 (Hour Pillar)

* **구하는 항목**: 시간(Hour Stem), 시지(Hour Branch)
* **산출 방법**:
  * **시지**: 하루 24시간을 12개 구간으로 나눔 (자시: 23:30~01:29, 축시: 01:30~03:29 등).
  * **시간 (시두법)**: 일간에 의해 결정됩니다.
    * `시간 Index = (일간 Index * 2 + 시지 Index) % 10` (시지 Index는 子=0 기준)
    * 예: 갑(甲)일의 자(子)시 -> 갑자(甲子)시

## 2. 분석 데이터 산출 (Analysis Data)

사주 원국(Original Chart)을 바탕으로 분석된 2차 데이터들입니다.

### 2.1 십신/육친 (Ten Gods / Sipsin)

* **설명**: 일간(나)과 다른 간지와의 관계를 10가지로 분류한 것.
* **구하는 방법**:
    1. **오행 비교**: 일간의 오행과 대상의 오행 간의 생극(生剋) 관계 파악.
        * 나와 같음 (비겁) / 내가 생함 (식상) / 내가 극함 (재성) / 나를 극함 (관성) / 나를 생함 (인성)
    2. **음양 비교**: 일간과 대상의 음양 일치 여부 파악.
  * **매핑표**:
    * 비견(동일/동일), 겁재(동일/다름)
    * 식신(생함/동일), 상관(생함/다름)
    * 편재(극함/동일), 정재(극함/다름)
    * 편관(극힘/동일), 정관(극힘/다름)
    * 편인(생김/동일), 정인(생김/다름)

### 2.2 12운성 (Twelve Life Stages)

* **설명**: 천간이 지지에서 얻는 에너지의 강약 상태 (포태법).
* **구하는 방법**:
  * 각 천간(10개)마다 지지(12개)에 대응하는 12단계 상태표를 사용.
  * 순서: 장생 - 목욕 - 관대 - 건록 - 제왕 - 쇠 - 병 - 사 - 묘 - 절 - 태 - 양
  * 양간(甲丙戊庚壬)은 순행, 음간(乙丁己辛癸)은 역행합니다.
  * *예: 갑(甲)목은 해(亥)에서 장생, 자(子)에서 목욕...*

### 2.3 12신살 (Twelve Divine Spirits / Sinsal)

* **설명**: 지지와 지지 사이의 관계에서 오는 길흉화복의 별.
* **구하는 방법**:
  * 기준: 연지(또는 일지)의 **삼합(三合)** 오행 국(局).
  * 삼합의 왕지(王地) 다음 글자를 기준으로 순서대로 매핑.
  * *예: 신자진(申子辰) 수국인 경우, 사(巳)가 겁살, 오(午)가 재살...*
  * 순서: 겁살 - 재살 - 천살 - 지살 - 년살 - 월살 - 망신살 - 장성살 - 반안살 - 역마 - 육해살 - 화개

### 2.4 지장간 (Hidden Stems / Jijanggan)

* **설명**: 지지 속에 숨겨져 있는 천간들 (초기, 중기, 정기).
* **구하는 방법**:
  * 각 지지마다 정해진 지장간 매핑 테이블 사용.
  * *예: 인(寅) -> 무(戊, 초기), 병(丙, 중기), 갑(甲, 정기)*
  * **사령(Commanding)**: 월지의 경우, 태어난 날짜가 절기로부터 며칠 지났는지에 따라 초기/중기/정기 중 주도하는 기운(사령신)을 판단.

### 2.5 공망 (Void / Gongmang)

* **설명**: 천간은 10개, 지지는 12개라 짝을 짓고 남는 2개의 지지.
* **구하는 방법**:
  * 일주(또는 연주) 기준.
  * `(일지 Index - 일간 Index)`를 계산하여 해당 순(旬)을 찾고, 짝이 없는 뒤의 두 지지를 도출.
  * *예: 갑자순(甲子旬)에서는 술(戌), 해(亥)가 공망.*

### 2.6 오행 분석 (Five Elements Calculation)

* **설명**: 사주 전체에서 목, 화, 토, 금, 수의 개수와 비율.
* **구하는 방법**:
  * 사주 8글자의 오행을 카운트.
  * 지장간에 있는 오행까지 고려하는 가중치 방식 또는 단순 글자 수 방식 사용. (현재는 글자 수 기반 + 월지 가중치 등)

### 2.7 기타 신살 (Additional Shinsals)

12신살 외에 일주/사주 전체를 기준으로 판별하는 특수 신살들입니다.

#### 2.7.1 파워 신살 (Power Shinsals)

| 신살 | 설명 | 판별 기준 |
|------|------|----------|
| **백호대살** | 혈광지사, 프로페셔널, 집중력 | 특정 일주 조합 (갑진, 을미, 병술, 정축, 무진, 임술, 계축) |
| **괴강살** | 우두머리, 결단력, 미남미녀 | 일주가 경진, 경술, 임진, 임술, 무술 중 하나 |
| **양인살** | 프로성, 강한 경쟁력, 칼 다루는 직업 | 양간 일간 + 제왕지 지지 조합 (갑-묘, 병-오, 무-오, 경-유, 임-자) |

#### 2.7.2 상호작용 신살 (Interaction Shinsals)

| 신살 | 설명 | 판별 기준 |
|------|------|----------|
| **귀문관살** | 천재성, 신경쇠약, 통찰력 | 지지 쌍: 진-해, 오-축, 사-술, 묘-신, 인-미, 자-유 |
| **원진살** | 애증, 원망, 불화 | 지지 쌍: 자-미, 축-오, 인-유, 묘-신, 진-해, 사-술 |

#### 2.7.3 귀인성 (Noble Stars)

| 신살 | 설명 | 판별 기준 |
|------|------|----------|
| **천을귀인** | 최고의 길신, 후원자, 위기 시 도움 | 일간 기준 특정 지지 매핑 |
| **문창귀인** | 학문, 창의력, 문서 능력 | 일간 기준 특정 지지 매핑 |

#### 2.7.4 구현 파일

* **계산 로직**: `lib/utils/saju_shinsal_calculator.dart`
* **UI 위젯**: `lib/widgets/shinsal_widget.dart`

### 2.8 형충회합 관계 분석 (Relationship Analysis)

지지 간의 상호작용 관계를 분석합니다. 합(合)은 조화와 결합을, 충(沖)은 갈등과 변화를 의미합니다.

#### 2.8.1 육합 (Six Combinations)

두 지지가 만나 새로운 오행으로 합화하는 관계입니다.

| 육합 | 합화 오행 | 의미 |
|------|----------|------|
| 子丑 (자축합) | 土 | 음양 조화, 안정적 결합 |
| 寅亥 (인해합) | 木 | 성장과 발전 |
| 卯戌 (묘술합) | 火 | 열정과 변화 |
| 辰酉 (진유합) | 金 | 결단과 성취 |
| 巳申 (사신합) | 水 | 지혜와 유연성 |
| 午未 (오미합) | 火 | 에너지 폭발 |

#### 2.8.2 삼합 (Three Combinations)

세 지지가 모여 하나의 오행 국(局)을 이루는 강력한 합입니다.

| 삼합 | 국(局) | 왕지 | 구성 |
|------|--------|------|------|
| 申子辰 | 水局 | 子 | 신-자-진 |
| 亥卯未 | 木局 | 卯 | 해-묘-미 |
| 寅午戌 | 火局 | 午 | 인-오-술 |
| 巳酉丑 | 金局 | 酉 | 사-유-축 |

* **반합**: 삼합 중 2글자만 있는 경우 (약한 합)

#### 2.8.3 방합 (Directional Combinations)

같은 방위/계절의 세 지지가 모이는 합입니다.

| 방합 | 방위 | 오행 | 계절 | 구성 |
|------|------|------|------|------|
| 寅卯辰 | 東方 | 木 | 봄 | 인-묘-진 |
| 巳午未 | 南方 | 火 | 여름 | 사-오-미 |
| 申酉戌 | 西方 | 金 | 가을 | 신-유-술 |
| 亥子丑 | 北方 | 水 | 겨울 | 해-자-축 |

#### 2.8.4 충 (Clashes)

서로 반대 방위에 있는 두 지지가 부딪치는 관계입니다.

| 충 | 의미 |
|------|------|
| 子午충 | 남북 충돌, 급격한 변화 |
| 丑未충 | 토의 충돌, 고집 대립 |
| 寅申충 | 동서 충돌, 활동적 변화 |
| 卯酉충 | 동서 충돌, 결단의 시기 |
| 辰戌충 | 토의 충돌, 창고 열림 |
| 巳亥충 | 수화 충돌, 극적 변화 |

#### 2.8.5 원진 (Distant Harm)

미워하면서도 끊을 수 없는 애증 관계입니다.

| 원진 | 관계 |
|------|------|
| 子未 | 자-미 원진 |
| 丑午 | 축-오 원진 |
| 寅酉 | 인-유 원진 |
| 卯申 | 묘-신 원진 |
| 辰亥 | 진-해 원진 |
| 巳戌 | 사-술 원진 |

#### 2.8.6 해 (Harm)

서로 해치는 관계로 은밀한 손상을 의미합니다.

| 해 | 관계 |
|------|------|
| 子未 | 자-미 해 |
| 丑午 | 축-오 해 |
| 寅巳 | 인-사 해 |
| 卯辰 | 묘-진 해 |
| 申亥 | 신-해 해 |
| 酉戌 | 유-술 해 |

#### 2.8.7 인접성 가중치

* **배열합 기준**: 지지의 명리학적 인접 관계 (자-축, 인-묘, 진-사, 오-미, 신-유, 술-해)
* 인접한 지지 간의 합은 더 강력하게 작용

#### 2.8.8 구현 파일

* **계산 로직**: `lib/utils/saju_calculator.dart` (`_calculateRelations()`)

## 3. 운세 데이터 (Fortune Data)

### 3.1 대운 (Daewoon)

* **설명**: 10년마다 바뀌는 큰 흐름의 운세.
* **구하는 방법**:
  * **기준**: 월주(Month Pillar).
  * **순행/역행**:
    * 남성 양년생(甲丙...) / 여성 음년생(乙丁...) -> **순행** (월주 다음 간지부터 순서대로)
    * 남성 음년생 / 여성 양년생 -> **역행** (월주 이전 간지부터 역순으로)
  * **대운수(Age)**: 태어난 날부터 다음(순행)/이전(역행) 절기까지의 날짜 수 / 3.

### 3.2 세운 (Yearly Fortune / Seun)

* **설명**: 매년 들어오는 연운.
* **구하는 방법**:
  * 해당 연도의 간지(2024년=甲辰)를 그대로 사용.
  * 매년 입춘을 기준으로 변경됨.

### 3.3 월운 (Monthly Fortune / Wolun)

* **설명**: 세운 내에서 매월 들어오는 월운.
* **구하는 방법**:
  * 해당 연도의 12개월 지지는 **고정** (1월=寅/丑, 2월=卯/寅... *기준에 따라 다름, 현재 앱은 양력 2월=寅월 기준*).
  * **월간**: 연간에 따른 월두법 재적용.
  * *중요*: 양력 1월은 입춘 전이므로 **전년도 연간**을 기준으로 월두법을 적용하여 1월(축월)의 간지를 산출함.

## 4. 만세력 조회 기능 (Manse-ryeol Lookup Feature)

사용자가 입력한 생년월일시 정보를 바탕으로 사주 원국과 대운/세운을 계산하여 시각적으로 보여주는 기능입니다.

### 4.1 입력 정보 (Input)

* **기본 정보**: 이름, 성별 (남/여)
* **생년월일**: 연, 월, 일 (양력/음력/윤달 구분 필수)
* **태어난 시간**: 시, 분
  * **시간 모름**: 입력하지 않을 경우 '모름'으로 처리하여 시주를 제외한 삼주(三柱)만 산출.
  * **자시 구분**: 야자시/조자시 옵션 또는 표준 자시(23:30~01:29) 적용.

### 4.2 출력 정보 (Output)

* **사주 원국표 (Four Pillars Chart)**
  * **구성**: 연주, 월주, 일주, 시주의 천간/지지 8글자 배치.
  * **시각화**: 오행 색상(목:청, 화:적, 토:황, 금:백, 수:흑)을 배경색이나 글자색으로 적용하여 직관적 파악 지원.
  * **보조 정보**: 각 간지별 십신(육친), 12운성, 12신살, 공망, 지장간 표시.

* **대운표 (Daewoon Chart)**
  * 10년 단위의 대운 흐름을 순서대로 나열.
  * 각 대운의 시작 나이(대운수) 및 간지, 십신, 12운성 표시.
  * **현재 대운 강조**: 사용자의 현재 나이에 해당하는 대운을 시각적으로 강조(Highlight).

* **세운/월운 조회**
  * 특정 연도(세운) 및 월(월운)을 선택하여 해당 시기의 간지와 12신살 등을 조회하는 기능.

* **오행 분석 그래프**
  * 사주 원국 내 오행(목, 화, 토, 금, 수)의 분포 비율을 그래프나 수치로 제공.

## 5. 당령 및 계절별 상생식 (Seasonal Mission & Strategy)

사용자의 생년월일을 바탕으로 **8품별 당령(Command)**을 계산하고, 계절별 상생식에 필요한 **5대 핵심 오행**이 사주 원국에 존재하는지 전수 조사하여 비즈니스 관점의 진단 데이터를 제공합니다.

### 5.1 8품별 당령 (Eight Grades Dangryeong)

* **설명**: 태어난 월/일을 기준으로 8개 품(品) 중 하나를 결정하고, 해당 품의 당령(사령) 천간을 부여.
* **구하는 방법**:
  * **양력 날짜 기준**으로 다음 범위에 따라 품과 당령 결정:

| 날짜 범위 | 당령 | 품명 | 계절 | 핵심 테마 |
|-----------|------|------|------|----------|
| 12/23 ~ 2/3 | 癸(계) | 자축품 | 겨울 | 유통과 저장 |
| 2/4 ~ 3/20 | 甲(갑) | 인묘품 | 봄 | 기획과 시작 |
| 3/21 ~ 5/4 | 乙(을) | 묘진품 | 봄 | 기획과 시작 |
| 5/5 ~ 6/21 | 丙(병) | 사오품 | 여름 | 확장과 성장 |
| 6/22 ~ 8/6 | 丁(정) | 오미품 | 여름 | 확장과 성장 |
| 8/7 ~ 9/22 | 庚(경) | 신유품 | 가을 | 생산과 제조 |
| 9/23 ~ 11/7 | 辛(신) | 유술품 | 가을 | 생산과 제조 |
| 11/8 ~ 12/22 | 壬(임) | 해자품 | 겨울 | 유통과 저장 |

### 5.2 계절별 5대 핵심 오행 (Seasonal Key Elements)

* **설명**: 각 계절에 따라 비즈니스 성공에 필요한 5대 요소(Product, Market, Customer, Skill, Trend)와 대응 천간을 매핑.
* **비즈니스 역할 매핑**:

#### 봄 (Spring: 인묘진월) - 목왕절, 기획과 시작
| 역할 | 대응 천간 | 의미 |
|------|----------|------|
| Product (상품/지식) | 癸, 壬 | 상품의 원천 기술, 특허, 아이디어, 교육 콘텐츠 |
| Market (시장/플랫폼) | 甲, 乙 | 교육 시장, 건축, 기획, 인적 네트워크 |
| Customer (고객/반응) | 丙, 丁 | 대중의 인기, 마케팅, 바이럴, 프레젠테이션 |
| Skill (전문성/조절) | 庚, 辛 | 가지치기(벽갑), 구조조정 능력, 컨설팅 |
| Trend (환경) | 戊 | 활동의 무대, 연고지 |

#### 여름 (Summer: 사오미월) - 화왕절, 확장과 성장
| 역할 | 대응 천간 | 의미 |
|------|----------|------|
| Product (상품/연료) | 甲, 乙 | 지속적인 아이템 공급, 투자금, 서포트 |
| Market (시장/확산) | 丙, 丁 | 방송, 언론, 통신, 서비스업, 화려한 조명 |
| Customer (고객/결실) | 庚, 辛 | 실질적인 매출, 현금화 능력, 구매력 |
| Skill (전문성/안정장치) | 壬, 癸 | 조후, 위기 관리 능력, 휴식, 힐링 |
| Trend (환경) | 戊, 己 | 뜨거운 열기를 저장하고 중개하는 플랫폼 |

#### 가을 (Autumn: 신유술월) - 금왕절, 생산과 제조
| 역할 | 대응 천간 | 의미 |
|------|----------|------|
| Product (상품/원자재) | 戊, 己 | 광산, 원석, 부동산, 1차 자원 |
| Market (시장/제조업) | 庚, 辛 | 공장, 제조 설비, 금융, 권력 기관 |
| Customer (고객/유통) | 壬, 癸 | 무역, 운송, 요식업, 해외 시장 연결 |
| Skill (전문성/가공) | 丁, 丙 | 제련, R&D, 자격증, 특수 기술 |
| Trend (환경) | 甲 | 금(도끼)이 쓰일 용도(나무), 시장의 수요 |

#### 겨울 (Winter: 해자축월) - 수왕절, 유통과 저장
| 역할 | 대응 천간 | 의미 |
|------|----------|------|
| Product (상품/브랜드) | 辛, 庚 | 명품, 브랜드, 종잣돈, 완성된 시스템 |
| Market (시장/유통망) | 壬, 癸 | 온라인 시장, 밤의 문화, 해외, 비밀스러운 거래 |
| Customer (고객/타겟) | 甲, 乙 | 새로운 시작을 준비하는 사람들, 교육생 |
| Skill (전문성/온기) | 丙, 丁 | 해동, 따뜻한 감성, 서비스 마인드, 상담 능력 |
| Trend (환경) | 戊 | 범람하는 물(시장)을 조절하는 댐, 토목, 부동산 |

### 5.3 사주 원국 스캔 (Saju Scanning)

* **설명**: 5대 핵심 요소의 대응 천간이 사주 원국 어디에 위치하는지 전수 조사.
* **스캔 위치**:
  1. **천간(Stems)**: 연간, 월간, 일간, 시간 (4개) - 가장 강력한 보유
  2. **지지(Branches)**: 연지, 월지, 일지, 시지 (4개) - 지지 본 오행 기준 매칭
  3. **지장간(Hidden Stems)**: 각 지지의 숨겨진 천간 - 잠재력으로 판단

### 5.4 상태 판정 (Status Determination)

* **Active (보유)**: 천간 또는 지지(본 오행 기준)에 존재 → 즉시 활용 가능한 무기
* **Potential (잠재)**: 지장간에만 존재 → 노력으로 개발 가능한 잠재 재능
* **Missing (미보유)**: 어디에도 존재하지 않음 → 협업/외부 자원으로 보완 필요

### 5.5 출력 정보 (Output)

* **당령 카드**: 당령 천간, 품명, 계절, 핵심 테마, 역할 설명
* **5대 요소 대시보드**: 가로 스크롤 카드 형태로 각 요소의 보유 상태 표시
  * 아이콘, 라벨, 타겟 천간, 상태 배지 (✅보유/⚠️잠재/❌미보유)
* **상세 분석 패널**: 보유/잠재/미보유 요소 목록 및 활용 가이드

### 5.6 8품별 희용 (Heeyong - Favorable Elements)

* **설명**: 각 품(계절)에서 조후(調候) 원리에 따라 균형을 맞추기 위해 필요로 하는 핵심 천간.
* **구하는 방법**:
  * 8품별로 정해진 희용 천간 2개를 매핑 테이블에서 조회
  * 사주 원국(천간, 지지, 지장간)에서 희용 천간 보유 여부 스캔

#### 8품별 희용 매핑 테이블

| 품(品) | 월령 | 당령 | 희용 | 이유 |
|--------|------|------|------|------|
| 자축품 | 子丑 | 癸水 | 丙, 甲 | 한랭한 水를 따뜻하게 할 火가 필요, 木이 火를 생하여 보조 |
| 인묘품 | 寅卯 | 甲木 | 癸, 丙 | 木의 성장을 위해 水의 자양분과 火의 태양이 필요 |
| 묘진품 | 卯辰 | 乙木 | 丙, 癸 | 火의 태양으로 꽃을 피우고, 水로 뿌리를 적셔야 함 |
| 사오품 | 巳午 | 丙火 | 癸, 乙 | 강한 火를 水로 조절하고, 木이 불의 연료가 됨 |
| 오미품 | 午未 | 丁火 | 壬, 庚 | 뜨거운 火를 큰 물(壬水)로 식히고, 金이 水를 생함 |
| 신유품 | 申酉 | 庚金 | 丁, 壬 | 金을 火로 제련하여 보석으로 만들고, 水로 설기 |
| 유술품 | 酉戌 | 辛金 | 壬, 丁 | 보석(辛金)을 水로 씻어 빛나게 하고, 火로 정교하게 다듬음 |
| 해자품 | 亥子 | 壬水 | 丁, 辛 | 차가운 水에 따뜻한 불빛(丁火)이 필요하고, 金이 水의 수원 |

#### 희용 상태 판정

* **Complete (완전 보유)**: 모든 희용 천간이 천간 또는 지지에 존재 → 녹색
* **Partial (일부 보유)**: 일부 희용 천간이 천간 또는 지지에 존재 → 파란색
* **Potential (잠재)**: 지장간에만 존재 → 주황색
* **Missing (미보유)**: 어디에도 존재하지 않음 → 빨간색

#### 희용 UI 출력

* **희용 카드**: 당령 카드 아래에 배치
  * 희용 천간 칩 (보유 상태별 색상/아이콘 표시)
  * 전체 희용 상태 배지 (완전 보유/일부 보유/잠재/미보유)
  * 해당 품의 희용이 필요한 이유 설명

### 5.7 8품별 배합/상생식 (Baehap - Partner Element)

* **설명**: 각 품(계절)에서 성공을 위해 필요로 하는 핵심 파트너 천간. 상생식(相生式)에 따라 당령과 배합이 만나면 완전한 성격(成格)이 됨.
* **구하는 방법**:
  * 8품별로 정해진 배합 천간 1개를 매핑 테이블에서 조회
  * 사주 원국(천간, 지지, 지장간) 및 **현재 대운**에서 배합 천간 보유 여부 스캔

#### 8품별 배합 매핑 테이블

| 품(品) | 당령 | 배합 | 배합 의미 | 미션/역할 |
|--------|------|------|----------|----------|
| 자축품 | 癸水 | 甲 | 갑목이 있어야 머릿속의 아이디어를 교육 커리큘럼이나 문화 콘텐츠로 구축 | 기획, 교육, 사상, 기초 학문, 콘텐츠 |
| 인묘품 | 甲木 | 癸 | 계수가 있어야 가르칠 지식 체계를 갖추게 됨 (준비된 선생) | 학교, 유치원, 교육, 복지, 건축 설계 |
| 묘진품 | 乙木 | 丙 | 병화가 있어야 조직 내 소통으로 공적 활동(커뮤니티 빌딩)을 수행 | 커뮤니티, 네트워킹, 소통, 공적 활동, 인맥 관리 |
| 사오품 | 丙火 | 乙 | 을목이 있어야 조직의 실체(인맥)를 구성하고 유연한 네트워크를 만듦 | 조직, 인맥, 네트워크, 마케팅, 확장 |
| 오미품 | 丁火 | 庚 | 경금이 있어야 기술로 제품을 제작할 수 있음 (실물 생산) | 기술, 제품, 제조, 생산, 품질 관리 |
| 신유품 | 庚金 | 丁 | 정화가 있어야 원석을 제련하여 도구나 상품으로 완성 | 제련, 가공, 품질, 전문성, 기술 연마 |
| 유술품 | 辛金 | 壬 | 임수가 있어야 완성된 상품이 시장에 유통됨 (판매 채널) | 유통, 판매, 시장, 무역, 해외 |
| 해자품 | 壬水 | 辛 | 신금이 있어야 시장에서 거래할 물건(브랜드, 상품)을 확보 | 상품, 브랜드, 자산, 투자, 거래 |

#### 배합 스캔 위치

1. **천간(Stems)**: 연간, 월간, 일간, 시간 - 원국에 직접 보유
2. **지지(Branches)**: 지지 본 오행 기준 매칭
3. **지장간(Hidden Stems)**: 잠재력으로 판단
4. **대운(Daewoon)**: 현재 대운의 천간/지지/지장간에서 배합 존재 여부 스캔

#### 배합 상태 판정

| 상태 | 조건 | UI 표시 | 설명 |
|------|------|---------|------|
| **성격(Complete)** | 천간 또는 지지에 존재 | ✅ 최고의 파트너! (녹색) | 원국에 배합 보유, 즉시 활용 가능 |
| **운에서 발복(Daewoon)** | 현재 대운에 존재 | 🔮 운에서 만남 (보라색) | 대운에서 배합을 만나 발복 시기 |
| **잠재(Potential)** | 지장간에만 존재 | ⚠️ 숨은 재능 (주황색) | 노력으로 개발 가능한 잠재력 |
| **미보유(Missing)** | 어디에도 없음 | ❌ WANTED (빨간색) | 협업/외부 자원으로 보완 필요 |

  * 동지 이전 출생 시 전년도 동지 기준 적용

#### 배합 UI 출력

* **배합 카드**: 희용 카드 아래에 배치
  * "성공 파트너 (배합)" 헤더 + 상태 배지

  * 위치 정보 (천간/지지/지장간/대운 중 어디에 있는지)

  * 배합 의미 설명 텍스트
  * 키워드 태그 (#기획 #교육 #사상 등)

## 6. 격국(格局) 판별 기능 (Gyeok Determination)

자평진전(子平眞詮) 규칙에 따라 사주의 핵심 정체성인 격국을 판별하고, 상신/구신을 분석하여 성격(成格) 여부를 판정합니다.

### 6.1 격국 판별 로직

* **설명**: 월지(月支)를 중심으로 사주의 핵심 격(格)을 확립.
* **판별 우선순위**:
  1. **합국(合局)** 우선 - 월지 포함 삼합/방합/반합
  2. **투출(透出)** 판단 - 월지 지장간이 천간에 나타나는지
  3. **월지 유형별 판정** - 사왕지/사생지/사고지
  4. **비견/겁재 예외** - 건록격/양인격/월겁

#### 월지 유형별 판정

| 유형 | 해당 지지 | 판정 규칙 |
|------|----------|----------|
| 사왕지 | 子午卯酉 | 월지 정기로 격 확정 (午월 丙丁 동시 투간 시 사령 구간 적용) |
| 사생지 | 寅申巳亥 | 사령+삼합+투간 3중 조건 충족 시 변격, 아니면 정기 유지 |
| 사고지 | 辰戌丑未 | 합국 우선, 특칙 적용, 투출 순, 정기 순 |

### 6.2 격국 분류

| 분류 | 해당 격국 | 운용 전략 |
|------|----------|----------|
| 사길격 | 정관격, 정재격, 편재격, 정인격, 식신격 | 순용(順用) |
| 사흉격 | 편관격, 상관격, 편인격, 양인격, 건록격 | 역용(逆用) |
| 파격 | 월겁 | 파격 수습 |

### 6.3 상신(相神) 계산

* **설명**: 격국을 돕는 용신(用神). 격을 완성하고 강화하는 십신.
* **구하는 방법**: 격국별 우선순위 테이블에 따라 천간(연간/월간/시간)에서 탐색.

#### 격국별 상신 우선순위

| 격국 | 상신 후보 | 역할 | 설명 |
|------|----------|------|------|
| 정관격 | 정재, 편재 | 재생관 | 재성이 정관을 생하여 벼슬을 높임 |
| 정관격 | 정인, 편인 | 호관 | 인성이 상관으로부터 정관을 보호 |
| 정재격/편재격 | 식신, 상관 | 식상생재 | 식상이 재성을 생하여 재물을 늘림 |
| 정인격 | 정관, 편관 | 관인상생 | 관성이 인성을 생하여 학문을 높임 |
| 식신격 | 정재, 편재 | 식신생재 | 식신의 기운이 재물로 유통됨 |
| 식신격 | 편관 | 식신제살 | 식신이 칠살을 제압하여 공을 세움 |
| 편관격 | 식신 | 식신제살 | 식신이 칠살을 제압하여 권위를 세움 |
| 편관격 | 정인, 편인 | 살인상생 | 인성이 칠살의 흉함을 완화 |
| 편관격 | 겁재 | 양인합살 | 양인이 칠살과 합하여 힘을 합침 |
| 상관격 | 정인, 편인 | 상관패인 | 인성이 상관을 제어하여 안정 |
| 상관격 | 정재, 편재 | 상관생재 | 상관의 기운이 재물로 유통됨 |
| 편인격 | 정재, 편재 | 재제편인 | 재성이 편인을 제어하여 균형 |
| 양인격/건록격 | 정관, 편관 | 관살제복 | 관살이 비겁을 제어하여 균형 |
| 양인격/건록격 | 식신, 상관 | 식상설기 | 식상이 강한 일간의 힘을 뺌 |

### 6.4 기신(忌神) 식별

* **설명**: 격국을 해치는 흉신. 격을 파괴하거나 약화시키는 십신.
* **구하는 방법**: 격국별 위협 테이블에 따라 천간에서 탐색.

#### 격국별 기신 매핑

| 격국 | 기신 | 위협 | 필요 구신 |
|------|------|------|----------|
| 정관격 | 상관 | 상관견관 | 정인, 편인 (인제상관) |
| 정관격 | 비견, 겁재 | 비겁쟁재 | 정관, 편관 (관제비겁) |
| 정재격 | 편인 | 도식 | 정재, 편재 (재제편인) |
| 정재격/편재격 | 비견, 겁재 | 비겁겁재 | 정관, 편관 (관제비겁) |
| 정인격 | 정재, 편재 | 재파인 | 비견, 겁재 (비제재성) |
| 식신격 | 편인 | 효신탈식 | 정재, 편재 (재제편인) |
| 편관격 | 편인 | 효신탈식 | 정재, 편재 (재제편인) |
| 상관격 | 정재, 편재 | 재파인 | 비견, 겁재 (비제재성) |

### 6.5 구신(救神) 계산

* **설명**: 기신을 제압하는 구원자. 파격을 막고 격을 보호하는 십신.
* **구하는 방법**:
  1. **극제(剋制)**: 기신을 극하는 십신 탐색
  2. **합거(合去)**: 기신과 천간합하여 무력화하는 글자 탐색

#### 천간합 매핑 (합거 판단용)

| 천간합 | 설명 |
|--------|------|
| 甲 ↔ 己 | 갑기합토 |
| 乙 ↔ 庚 | 을경합금 |
| 丙 ↔ 辛 | 병신합수 |
| 丁 ↔ 壬 | 정임합목 |
| 戊 ↔ 癸 | 무계합화 |

### 6.6 성격(成格) 판정

* **설명**: 격국이 완성되었는지 판정.
* **판정 기준**:

| 조건 | 판정 결과 | 설명 |
|------|----------|------|
| 상신 O, 기신 X | 성격(成格) | 상신이 격을 도움, 흉신 부재 |
| 상신 O, 기신 O, 구신 O | 성격(成格) | 기신을 구신이 제압하여 격 보호 |
| 상신 O, 기신 O, 구신 X | 파격(破格) 위기 | 기신이 격을 해치나 구신 부재 |
| 상신 X, 기신 X | 보통격 | 상신/기신 모두 부재 |
| 상신 X, 기신 O | 불완전격 | 상신 부재로 격이 완성되지 않음 |

### 6.7 신강/신약 판정

* **설명**: 일간의 세력 강약 판정.
* **구하는 방법**: 비겁(비견/겁재) + 인성(정인/편인) 점수 합산.

#### 세력 점수 배점

| 위치 | 점수 |
|------|------|
| 월지 | 35점 |
| 일지 | 15점 |
| 기타 (연간, 월간, 시간, 연지, 시지) | 각 10점 |

* **판정**: 총점 50점 이상 → 신강, 50점 미만 → 신약

### 6.8 출력 정보

* **격국 카드**:
  * 격국명 (예: 정관격, 편재격)
  * 분류 뱃지 (사길격: 안정지향 / 사흉격: 도전지향 / 파격)
  * 신강/신약 뱃지
  * 성격 키워드 (원리원칙, 추진력 등)
  * 추천 분야 (공무원, 사업 등)
  * 운용 전략 (순용/역용)

* **격국 성립 원리 (확장 패널)**:
  * 월지(사회궁)
  * 합국 유형 또는 투출/정기 정보
  * 적용 규칙
  * 일간 세력 점수

* **상신/구신 분석 (확장 패널)**:
  * 성격 판정 상태 (성격/파격 위기/보통격/불완전격)
  * 상신 정보 (위치, 십신, 역할, 설명)
  * 기신 정보 (위치, 십신, 위협)
  * 구신 정보 (위치, 십신, 역할, 방법)

## 7. 기신(忌神) 종합 산출

기신(忌神)은 사주 원국에서 용신(用神)을 극(剋)하거나 격국을 파격으로 만드는 흉신을 의미합니다. 본 엔진은 **격국론 + 억부론 + 조후론** 3가지 관점에서 기신을 종합적으로 산출합니다.

### 7.1 격국론 기반 기신

* **정의**: 격국이나 상신(相神)을 파괴하는 글자
* **산출 방법**:
  * `json/gyeokguk_sangsin_map.json`에서 격국별 기신 매핑 조회
  * `gyeok_gisin`: 격국을 직접 파괴하는 십신
  * `sangsin_gisin`: 상신을 손상시키는 십신
  * 원국 천간에서 해당 십신 존재 여부 확인

**예시**:
| 격국 | 격국 기신 | 상신 기신 | 구신(救神) |
|------|----------|----------|----------|
| 정관격 | 상관 | 겁재 | 정인 |
| 식신격 | 편인 | 편관 | 편재 |
| 양인격 | 식신 | 편재 | 편인 |

### 7.2 억부론 기반 기신

* **정의**: 일간의 중화를 방해하는 글자
* **산출 방법**:
  * `json/eokbu_gisin_rules.json`에서 규칙 조회
  * **신강 사주**: 인성(정인, 편인), 비겁(비견, 겁재)이 기신
    - 이유: 이미 강한 일간을 더 강하게 하여 균형을 깨뜨림
  * **신약 사주**: 식상(식신, 상관), 재성(정재, 편재), 관성(정관, 편관)이 기신
    - 이유: 약한 일간의 힘을 더 빼앗아 균형을 깨뜨림

### 7.3 조후론 기반 기신

* **정의**: 계절적 기후 균형을 해치는 오행
* **산출 방법**:
  * `json/johu_gisin_rules.json`에서 규칙 조회
  * **동월생 (11, 12, 1월)**: 수(水), 금(金)이 기신
    - 이유: 이미 추운 계절에 더 춥게 만드는 오행
  * **하월생 (5, 6, 7월)**: 화(火), 목(木)이 기신
    - 이유: 이미 뜨거운 계절에 더 뜨겁게 만드는 오행

### 7.4 기신 해석

* **데이터 소스**: `json/gisin_interpretations.json`
* **십성별 해석**:

| 십성 | 부정적 영향 | 긍정적 측면 | 극복 전략 |
|------|-----------|-----------|---------|
| 비견 | 재물 손실, 경쟁 패배, 배신 | 경쟁을 통한 성장 | 협력을 배우세요 |
| 겁재 | 재물 탈취, 배우자 갈등 | 결단력, 추진력 향상 | 욕심을 줄이세요 |
| 식신 | 밥그릇 위협, 활동력 저하 | 창의성 내면화 | 때를 기다리세요 |
| 상관 | 구설수, 실직, 관재수 | 창의적 돌파 | 말을 아끼세요 |
| 정재 | 학업 중단, 고부 갈등 | 실리 추구 | 안정을 추구하세요 |
| 편재 | 기획력 약화, 건강 문제 | 현실적 감각 | 건강 관리하세요 |
| 정관 | 스트레스, 위축, 직장 부적응 | 책임감 학습 | 자신을 억누르지 마세요 |
| 편관 | 사고, 질병, 소송 | 위기 대응력 향상 | 신중하게 행동하세요 |
| 정인 | 게으름, 의존성 증가 | 학습과 사색 | 행동으로 옮기세요 |
| 편인 | 도식, 고립 | 독창적 사고 | 소통을 늘리세요 |

  2. 개수가 가장 적은 오행

| **금(Metal)** | 흰색, 회색, 금색 | 4, 9 | 서쪽 | 금속 액세서리, 시계, 거울 | 정리정돈, 규칙적인 생활, 결단 |
